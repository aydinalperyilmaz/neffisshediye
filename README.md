# Neffiss Hediye

Çiçek ve hediye teslimat platformu — Şişli / Çağlayan bölgesi.

## Paneller

- `index.html` — Müşteri paneli (çiçek sipariş sayfası)
- `logingiris.html` — Çiçekçi paneli (sipariş yönetimi, alarm sistemi)
- `admin.html` — Yönetici paneli (çiçekçi ve sipariş yönetimi)

## Altyapı

- Backend: Supabase (`fnfczglxcnqgrdiqfuig`)
- Ödeme: PayTR (iFrame entegrasyonu)
- Hosting: GitHub Pages + Cloudflare DNS
- Domain: neffisshediye.com
